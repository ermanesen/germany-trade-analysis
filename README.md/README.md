# Junior Trade Data Analysis – Germany

## Overview
This project presents a junior-level data analysis of Germany’s trade balance
using publicly available World Bank data.

## Data Sources
- World Bank – Exports of goods and services (% of GDP)
- World Bank – Imports of goods and services (% of GDP)
- World Bank – GDP growth (annual %)

## Methodology
- Data cleaning and transformation
- Trade balance calculation
- Time-series visualization
- Correlation analysis
- Introductory OLS regression

## Key Findings
- Germany maintains a structurally positive trade balance.
- Global crises caused temporary declines.
- GDP growth influences trade balance dynamics.

## Tools
- Python
- pandas
- matplotlib
- seaborn
- statsmodels
